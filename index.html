Learning to use Github.

>git init
Initialising git.
>git add 'filename'
Add the file to a staging area.
>git commit -m "message"
Commits the changes in the staging area. -m to add a message for the commit.
>git log
Displays all commits.
>git clone 'git repository link'
Pulls all files from a git repository to your current directory.
>git push
Pushes all commits to git repository.
>git config --list
Lists all config details.
